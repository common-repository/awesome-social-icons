== Description ==

<ul>
 <li>Type of icons round and square</li>
<li> Animated Rotation Effect in  Icons.</li>
<li>Shadow in icons.</li>
<li>Different size of social icons. </li>
<li>Widget and Shortcode </li>
</ul>

Awesome Social Icons is providing two type of icons square and round, further there is also <strong>animation effect which looks much attractive</strong>.

== Installation ==

1. Upload the entire awesomesocial folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings menu under this, there will be the Awesome Social Menu (Settings>Awesome Social).
4. Configure the settings putting the URLs to your various social profiles, and if you want to show animation on hover, shadow,open link in new tab and size, configure it in Awesome Social Settings.
5. Shortcode is [awesome_social_icon].

== Frequently Asked Questions ==

= How to make icons animated ? =

Go to Settings>Awesome Social from left side admin menu bar, then click on Settings tab. Then click on Animation on hover, text box will be appear and then enter value in seconds,  like just enter 0.5 and it will give the delay of 0.5 seconds in animation.

= Don't want shadow in icons ? =

In settings simply checked " Don't use shadow in icons.", by default its unchecked.
